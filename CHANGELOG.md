# 更新日志

## [1.0.0] - 2024-04-10

### 新增
- ✨ 初始版本发布
- 🎨 像素魏征动画（大眼萌造型）
- 💬 气泡对话框（多句话轮播）
- 🔌 OpenClaw Skill 集成
- 🖥️ 后台运行支持（无窗口）
- 🌐 HTTP API（talk/stop/status）
- 🖼️ RDP 远程桌面兼容
- 📝 详细文档

### 功能
- 8帧待机动画（呼吸+眨眼）
- 8帧说话动画（嘴巴张合）
- 多句话轮播显示（每句3秒）
- 进度点指示
- 自动触发词检测
- 系统托盘模式（可选）

### 技术
- tkinter + PIL 像素动画
- Win32 API RDP 兼容
- HTTP RESTful API
- CLI 工具
- OpenClaw Hook
